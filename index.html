프로젝트 개요

기말 프로젝트

주요 업무

  • 웹 디자인 및 웹 기능 구현
  • 사용자가 원하는 컨텐츠를 선택 후 데이터를 입력하면 QR 생성해주는 생성기
  • H2 DB를 통해 사용자가 만든 컨텐츠 및 큐알을 DB에 저장 후 폴더에 업로드
  • H2 DB를 통해 미니게임(퀴즈) 컨텐츠 결과를 DB에 저장

기술 스택

Spring Boot, H2 DB, HTML5, JavaScript, CSS3

느낀 점

  • 사용자가 만든 이미지나 동영상을 폴더에 업로드 할 때 한글이름이나 특수 문자까지 생각하여 파일 이름을 지정 및 관리해야 오류가 안나고 이름이 저장될 때 파일 이름이 깨지지 않음.
  • 이미지나 동영상을 파일크기 제한을 두어서 관리를 해야 서버에 무리가 가지 않음을 깨달음
  • 사용자가 동영상을 이미 동영상을 중간부터 본 뒤에 업로드 할 때 QR 생성후 스캔해서 동영상을 실행할 때 중간부터 실행되는 문제가 발생 할 수 있어서 동영상 재생할 때 처음부터 재생 할 수 있게 잘 설정해야함.
