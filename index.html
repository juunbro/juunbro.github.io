프로젝트 개요

이 시스템은 교육 기관에서 강의실 좌석 예약의 혼잡 문제를 해결하기 위해 개발되었습니다. 특히, PC 사용이 필수적인 전공이나 실습 과목에서는 체계적인 좌석 관리가 중요하지만 대부분의 교육 기관에는 이러한 시스템이 부족합니다. 이 좌석 예약 시스템은 사용자가 모바일이나 웹으로 간편하게 강의실 좌석을 예약하고 실시간으로 예약 상황을 확인할 수 있도록 지원합니다. 이를 통해 학생의 편의를 증진하고 학업 효율을 극대화하며 관리자에게는 강의실 자원의 효율적인 활용과 체계적인 관리 기능을 제공합니다.

프로젝트 개요

기말 프로젝트로 진행한 QR 코드 생성기는 사용자가 원하는 다양한 컨텐츠를 QR 코드로 생성하고 관리할 수 있는 웹 애플리케이션입니다.

기술 스택

Spring Boot H2 DB HTML5 JavaScript CSS3

주요 기능

다양한 컨텐츠 타입 지원
  • 텍스트
  • 이미지
  • 비디오
  • 주소/URL
  • 연락처(vCard)
  • 위치 정보
  • 미니게임(퀴즈)
QR 코드 생성
  • 각 컨텐츠에 대한 고유 QR 코드 자동 생성
  • Base64 인코딩 지원
  • 안전한 파일명 처리
미니게임(퀴즈) 기능
  • 다양한 주제의 퀴즈 제공
  • 실시간 점수 계산
  • 상위 10위까지 랭킹 시스템
  • 모바일 친화적 UI

주요 API

컨텐츠 관리
  • POST /upload - 새로운 컨텐츠 업로드 및 QR 코드 생성
  • GET /view/{id} - 컨텐츠 조회
  • GET /uploads/{filename} - 업로드된 파일 조회
퀴즈 게임
  • GET /QuizJun - 퀴즈 게임 페이지
  • POST /api/quiz/score - 퀴즈 점수 저장
  • GET /api/quiz/leaderboard - 랭킹 조회

느낀 점

기술적 고려사항
  • 사용자가 만든 이미지나 동영상을 폴더에 업로드 할 때 한글이름이나 특수 문자까지 생각하여 파일 이름을 지정 및 관리해야 오류가 안나고 이름이 저장될 때 파일 이름이 깨지지 않음.
  • 이미지나 동영상을 파일크기 제한을 두어서 관리를 해야 서버에 무리가 가지 않음을 깨달음
  • 사용자가 동영상을 이미 동영상을 중간부터 본 뒤에 업로드 할 때 QR 생성후 스캔해서 동영상을 실행할 때 중간부터 실행되는 문제가 발생 할 수 있어서 동영상 재생할 때 처음부터 재생 할 수 있게 잘 설정해야함.

정렬 알고리즘 시각화

프로젝트 개요

다양한 정렬 알고리즘을 시각적으로 이해하고 학습할 수 있는 웹 애플리케이션입니다. React를 활용한 프론트엔드 개발 경험을 쌓기 위해 진행한 개인 프로젝트입니다.

주요 기능

정렬 시각화
  • 사용자가 원하는 정렬 선택을 하고 숫자를 입력
  • 정렬되는 과정을 한눈에 확인
  • 정렬될 때 사용되는 변수 확인과 변수 설명 확인 가능
정렬 알고리즘
  • 선택 정렬
  • 삽입 정렬
  • 버블 정렬
  • 병합 정렬
  • 힙 정렬
  • 트리 정렬

상세 기능

시각화 기능
  • 실시간 정렬 과정 시각화
  • 단계별 상세 설명
  • 변수 상태 추적
  • 활성 요소 하이라이팅
  • 정렬 속도 조절
  • 일시정지/재생 기능
  • 단계별 이동 기능
메인 화면
  • 정렬 알고리즘 선택
  • 배열 입력/수정
  • 정렬 과정 시각화
  • 단계별 설명
  • 변수 상태 표시
컨트롤 패널
  • 정렬 시작/정지
  • 속도 조절
  • 단계 이동
  • 배열 수정

기술 스택

Spring Boot React

느낀 점

학습 내용
  • 다양한 정렬 알고리즘에 대한 이해도 향상
  • React 사용 경험을 통한 프론트엔드 역량 강화
  • CORS 설정의 중요성을 깨달았으며 이를 올바르게 설정하지 않으면 예상치 못한 오류가 발생 할 수 있음을 경험했음.
