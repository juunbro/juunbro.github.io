프로젝트 개요

기말 프로젝트로 진행한 QR 코드 생성기는 사용자가 원하는 다양한 컨텐츠를 QR 코드로 생성하고 관리할 수 있는 웹 애플리케이션입니다.

기술 스택

Spring Boot H2 DB HTML5 JavaScript CSS3

주요 기능

다양한 컨텐츠 타입 지원
  • 텍스트
  • 이미지
  • 비디오
  • 주소/URL
  • 연락처(vCard)
  • 위치 정보
  • 미니게임(퀴즈)
QR 코드 생성
  • 각 컨텐츠에 대한 고유 QR 코드 자동 생성
  • Base64 인코딩 지원
  • 안전한 파일명 처리
미니게임(퀴즈) 기능
  • 다양한 주제의 퀴즈 제공
  • 실시간 점수 계산
  • 상위 10위까지 랭킹 시스템
  • 모바일 친화적 UI

주요 API

컨텐츠 관리
  • POST /upload - 새로운 컨텐츠 업로드 및 QR 코드 생성
  • GET /view/{id} - 컨텐츠 조회
  • GET /uploads/{filename} - 업로드된 파일 조회
퀴즈 게임
  • GET /QuizJun - 퀴즈 게임 페이지
  • POST /api/quiz/score - 퀴즈 점수 저장
  • GET /api/quiz/leaderboard - 랭킹 조회

느낀 점

기술적 고려사항
  • 사용자가 만든 이미지나 동영상을 폴더에 업로드 할 때 한글이름이나 특수 문자까지 생각하여 파일 이름을 지정 및 관리해야 오류가 안나고 이름이 저장될 때 파일 이름이 깨지지 않음.
  • 이미지나 동영상을 파일크기 제한을 두어서 관리를 해야 서버에 무리가 가지 않음을 깨달음
  • 사용자가 동영상을 이미 동영상을 중간부터 본 뒤에 업로드 할 때 QR 생성후 스캔해서 동영상을 실행할 때 중간부터 실행할 수 있게 잘 설정해야함.
