정렬 알고리즘 시각화 프로젝트 (개인 프로젝트)

정렬 알고리즘 시각화

프로젝트 개요

다양한 정렬 알고리즘을 시각적으로 이해하고 학습할 수 있는 웹 애플리케이션입니다. React를 활용한 프론트엔드 개발 경험을 쌓기 위해 진행한 개인 프로젝트입니다.

주요 기능

정렬 시각화
  • 사용자가 원하는 정렬 선택을 하고 숫자를 입력
  • 정렬되는 과정을 한눈에 확인
  • 정렬될 때 사용되는 변수 확인과 변수 설명 확인 가능
정렬 알고리즘
  • 선택 정렬
  • 삽입 정렬
  • 버블 정렬
  • 병합 정렬
  • 힙 정렬
  • 트리 정렬

상세 기능

시각화 기능
  • 실시간 정렬 과정 시각화
  • 단계별 상세 설명
  • 변수 상태 추적
  • 활성 요소 하이라이팅
  • 정렬 속도 조절
  • 일시정지/재생 기능
  • 단계별 이동 기능
메인 화면
  • 정렬 알고리즘 선택
  • 배열 입력/수정
  • 정렬 과정 시각화
  • 단계별 설명
  • 변수 상태 표시
컨트롤 패널
  • 정렬 시작/정지
  • 속도 조절
  • 단계 이동
  • 배열 수정

기술 스택

Spring Boot React

느낀 점

학습 내용
  • 다양한 정렬 알고리즘에 대한 이해도 향상
  • React 사용 경험을 통한 프론트엔드 역량 강화
  • CORS 설정의 중요성을 깨달았으며 이를 올바르게 설정하지 않으면 예상치 못한 오류가 발생 할 수 있음을 경험했음.
