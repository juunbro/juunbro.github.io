프로젝트 개요

WillGO는 위치 기반의 목표 달성을 돕는 서비스입니다. 사용자가 설정한 목표 장소에 방문하여 GPS를 통해 자동으로 인증하고 포인트 시스템으로 동기부여를 강화하는 위치기반 목표인증 서비스입니다.

느낀 점

기술적 성장
  • 카카오 로컬 API 키의 Github 노출 위험성을 인지하고 application-secret.properties 파일 분리 및 .gitignore 설정으로 API 키를 환경변수로 관리하여 보안 강화를 해보았다.
  • Swagger API 문서화 시 별도 폴더(Swagger/*)로 관리하던 중 Controller 매핑 중복 문제를 발견하고 실제 Controller에 직접 @Tag, @Operation 어노테이션을 적용하는 방식을 학습하고 전환하여 오류를 해결하고 코드 유지보수성 향상시켰다.
  • Oracle Server와 Ubuntu를 활용하여 서버를 구축하고 Github Actions를 통한 CI/CD 파이프라인을 구현하여 main 브랜치 push 시 자동으로 배포되는 파이프라인 시스템을 구축함으로써 서버 실전 상황과 배포 과정에서 발생할 수 있는 문제를 해결하는 방법을 배웠다.
